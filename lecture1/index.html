Complex Plane

Many applications in science and engineering model phenomena as a two-dimensional (2D) vector in a plane.

We can express the vector $z$ in a couple of ways:

Cartesian$z = (x, y)$
Polar$z = A \angle \theta$
where $$x = A \cos(\theta)$$ $$y = A \sin(\theta)$$

[Discussion of applications here, e.g., electrical circuits/impedance.]

Another useful way to express $z$ is as a complex number: $$z = x+iy$$

Here, $i$ represents the vertical-axis in the plane. We'll see that if we assign $i$ a special value, we can do useful mathematical operations on vectors in this 2D-space. It turns out that this special value is the imaginary unit, $i=\sqrt{-1}$. We'll see why.

Addition

Addition (or subtraction) of vectors—and thus addition of complex numbers—is straightforward. Just add the $x$-axis coordinates to get the new $x$-coordinate; and add the $y$-axis coordinates to get the new $y$-coordinate: $$z_1 = x + iy$$ $$z_2 = a + ib$$ $$z_1 + z_2 = (x + iy) + (a + ib) = (x+a) + i(y+b)$$
